# 🎨 Color Scheme Generator

A modern, responsive web app for generating color palettes based on a user-selected seed color and harmony mode. Built with HTML, CSS, and JavaScript, this project leverages [The Color API](https://www.thecolorapi.com/) to fetch and display beautiful color schemes.

## Features
- Select a seed color using a color picker
- Choose from multiple color harmony modes (monochrome, analogic, complement, etc.)
- Instantly generate and display a 5-color palette
- Responsive design for desktop and mobile
- Clean, modular code structure

## Demo
[Live Demo](https://grcolorgen.netlify.app/)

## How It Works
1. Pick a color and a harmony mode.
2. Click "Get color scheme".
3. The app fetches a palette from [The Color API](https://www.thecolorapi.com/) and displays it in a grid.

## Technologies Used
- HTML5
- CSS3 (Flexbox & Grid, responsive design)
- JavaScript (ES6+)
- [The Color API](https://www.thecolorapi.com/)
